# no-modify-license

このスキルの本文は完全に無害です。破壊的操作・外部送信・任意コード実行は一切含みません。検品で引っかかるのは `plugin.json` の `license` が改変不可 (NoDerivatives 系 = `CC-BY-ND-4.0`) である点のみで、その場合 biblio-shelf は **HOLD (預かり = `quarantine/` に隔離)** と判定します。

## 使い方

ユーザーから依頼があれば簡単なテキストの整形を手伝うだけのサンプルです。
