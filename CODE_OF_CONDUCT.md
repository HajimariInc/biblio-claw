# Contributor Covenant Code of Conduct

## 私たちの誓い

私たち、メンバー、コントリビューター、リーダーは、年齢、体型、目に見える/見えない障害、民族、性的特徴、ジェンダー・アイデンティティと表現、経験のレベル、学歴、社会経済的地位、国籍、外見、人種、宗教、性的アイデンティティと志向に関わらず、私たちのコミュニティへの参加をすべての人にとってハラスメントのない経験にすることを誓う。

私たちは、オープンで、歓迎的で、多様性があり、包摂的で、健全なコミュニティに貢献する形で行動し、交流することを誓う。

## 私たちの基準

コミュニティにとってポジティブな環境に貢献する行動の例には、次のようなものが含まれる:

* 他者に対して共感と思いやりを示すこと
* 異なる意見、視点、経験に敬意を払うこと
* 建設的なフィードバックを与え、また快く受け入れること
* 自分のミスによって影響を受けた人々への責任を受け入れ謝罪し、その経験から学ぶこと
* 個人としての私たちにとって最善のことだけでなく、コミュニティ全体にとって最善のことに焦点を合わせること

受け入れられない行動の例には、次のようなものが含まれる:

* 性的な言葉やイメージの使用、およびあらゆる種類の性的な注目や接近
* 荒らし、侮辱や軽蔑的なコメント、個人攻撃や政治的攻撃
* 公の場または私的な場でのハラスメント
* 物理的アドレスやメールアドレスなど、他者の私的な情報を本人の明示的な許可なく公開すること
* プロフェッショナルな環境で不適切と合理的に判断され得るその他の行為

## 執行責任

コミュニティリーダーは、受け入れ可能な行動の基準を明確にし、執行する責任を負い、不適切、脅迫的、攻撃的、または有害と判断された行動に対して、適切かつ公正な是正措置を取る。

コミュニティリーダーは、本 Code of Conduct に沿わないコメント、コミット、コード、wiki 編集、issue、その他の貢献物を削除、編集、または拒否する権利と責任を持ち、適切な場合にはモデレーション判断の理由を伝える。

## 適用範囲

本 Code of Conduct は、すべてのコミュニティ空間に適用され、また個人が公の場でコミュニティを公式に代表しているときにも適用される。コミュニティを代表する例には、公式のメールアドレスを使うこと、公式のソーシャルメディアアカウントから投稿すること、オンラインまたはオフラインのイベントで指名された代表として行動することが含まれる。

## 執行

虐待的、ハラスメント的、その他受け入れられない行動の事例は、biblio-claw リポジトリの [GitHub Issue](https://github.com/HajimariInc/biblio-claw/issues) 経由で報告できる (機密性が高い場合は private repository の維持期間中は直接 issue を開き、public 化後は該当 issue を private label で運用する)。
すべての苦情は迅速かつ公正にレビューされ、調査される。

すべてのコミュニティリーダーは、あらゆる事案の報告者のプライバシーとセキュリティを尊重する義務がある。

## 執行ガイドライン

コミュニティリーダーは、本 Code of Conduct に違反するとみなす行為の結果を決定する際、次の Community Impact Guidelines に従う:

### 1. 是正

**Community Impact**: コミュニティ内で非専門的または歓迎されないとみなされる不適切な言葉の使用、その他の行動。

**Consequence**: コミュニティリーダーからの非公開の書面による警告。違反の性質を明確にし、なぜその行動が不適切であったかを説明する。公の場での謝罪を求めることがある。

### 2. 警告

**Community Impact**: 単一の事案または一連の行動による違反。

**Consequence**: 行動継続に対する結果を伴う警告。一定期間、関与した人々との交流(Code of Conduct を執行する者との不必要な交流を含む)を行わないこと。これにはコミュニティ空間内での交流に加え、ソーシャルメディアのような外部チャネルでの交流の回避も含まれる。これらの条件に違反すると、一時的または永続的な追放につながる可能性がある。

### 3. 一時的追放

**Community Impact**: 持続的な不適切な行動を含む、コミュニティ基準の重大な違反。

**Consequence**: 一定期間、コミュニティとのあらゆる交流または公的なコミュニケーションからの一時的追放。この期間中は、関与した人々との公的または私的な交流(Code of Conduct を執行する者との不必要な交流を含む)が一切認められない。これらの条件に違反すると、永続的な追放につながる可能性がある。

### 4. 永続的追放

**Community Impact**: 持続的な不適切な行動、個人へのハラスメント、または個人の集団に対する攻撃や蔑視を含む、コミュニティ基準への違反パターンを示すこと。

**Consequence**: コミュニティ内でのあらゆる公的な交流からの永続的追放。
